# Agent Skills

[![skills.sh](https://skills.sh/b/muhammadrivaldy/agent-skills)](https://skills.sh/muhammadrivaldy/agent-skills)

Collection of agent skills for AI coding assistants. Install with one command:

```bash
npx skills add muhammadrivaldy/agent-skills
```

Or clone directly:

```bash
git clone https://github.com/muhammadrivaldy/agent-skills.git ~/.agents/skills
```
